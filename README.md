# Food Website

A responsive food website built with HTML and CSS showcasing various dishes and restaurant information.

## Features

- Home page with food showcase
- About page with restaurant information
- Menu page displaying food items
- Social media integration
- Responsive design

## Files Structure

- `website.html` / `home.html` - Main homepage
- `about.html` - About page
- `menu.html` - Menu page
- `socialmedia.html` - Social media page
- `custom.css` - Custom styles
- `normalize.css` - CSS reset/normalize
- Various image files for food items and backgrounds

## How to Run

1. Clone this repository
2. Open `website.html` or `home.html` in your web browser
3. Navigate through the different pages using the menu

## Technologies Used

- HTML5
- CSS3
- Responsive Design
